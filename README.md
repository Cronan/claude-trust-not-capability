# claude-code-skills

Skills for Claude Code. Copy a skill directory into `.claude/skills/` in your repo and invoke it by name.

```
cp -r skills/deep-init/ your-repo/.claude/skills/deep-init/
```

## Skills

| Skill | What it does |
|---|---|
| [`/deep-init`](skills/deep-init/) | Audit repo history, rebuild CLAUDE.md |
| [`/stet`](skills/stet/) | Strip AI writing patterns from text |
